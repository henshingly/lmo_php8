Team Plan

  1. Systemvoraussetzung
  2. Konfiguration
  3. Einbindung
  4. Download
  5. Versionsverlauf

Ermöglicht die Ausgabe im Link Spielpläne des LMO in einer PDF Ausgabe. Hier werden die gezeigten Partien (der ganzen Session ) einer ausgewählten Mannschaft als PDF-Dokument dargestellt.

Systemvoraussetzung

  • Lauffähiger LMO 4
  • Installiertes ClasslibAddon (c) von Tim Schumacher mit
    PDF-Class Wayne Munro, R&OS Ltd

Konfiguration

Das TeamPlan-Addon lässt sich über die Addonverwaltung des LMO konfigurieren.

Addonverwaltung

Die Konfiguration lässt sich in der Addonverwaltung des LMO vornehmen, klicken sie dazu auf den Punkt pdf. Folgende grundlegende Parameter lassen sich steuern:

linktarget
Verlinkung des PDF-Dokuments in ein neues Fenster ( z.B _blank )
show_adobe_link
Hinweis zum PDF-Reader mit Bild ==> 0 / 1
homepage
Anzeige eines Textes am Rand z.B. Ihre HP Adresse
anzeige
Ausgabe Sortiert nach Spieltag / Datum ==> 0 / 1
format
Ausgabeformat A4 Quer- / Hochformat ==> 0 / 1
teamfett
Hervorheben der Ausgewählten Mannschaft aus / an ==> 0 / 1
teamnamen
Anzeige Teamname lang / kurz / mittel ==> 0 / 1 / 2
serie
Getrennte Anzeige Hin- / Rückrunde 1 Seite / 2 Seiten ==> 0 / 1
