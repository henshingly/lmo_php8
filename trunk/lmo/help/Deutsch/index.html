Changelog

LMO 4.0.2 (Oktober 2009)

Features

  • editing of archived leagues in admin area
  • more improvements for installer
  • enhanced league stats
  • move matches to other matchdays (LMO Rounds Hack by timme)
  • Viewer displays all teams in a league if no fav team is selected
  • updated documentation

Translation

  • translation tool
  • romanian translation (arthur)
  • spanish translation of the installer (Gabriel)

LMO 4.0.1 (April 2009)

Features

  • new installer (should work with more servers)
  • Team note for Minitab (http://www.liga-manager-online.de/lmo-forum/viewtopic.php?t=7969)
  • Countdown for Mininext (plastic)
  • more graphic formats for icons (png, gif, jpg)
  • enhanced league stats (timme - http://www.liga-manager-online.de/lmo-forum/viewtopic.php?p=47571#47571)
  • updated documentation

Translation

  • spanish version of player stats, prediction game, ticker (mike73)
  • updated dutch translation (Pascal Wilkens)
  • updated french translation (mike73)
