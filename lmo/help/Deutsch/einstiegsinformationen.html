1. Einleitung

Herzlich Willkommen zum neuen Liga Manager Online. Mit diesem Skript können Sie Ihren Besuchern einen Ergebnisdienst über Ihre Lieblingsligen anbieten. Die gesamte Verwaltung der Ligen läuft dabei direkt auf dem Server ab; Sie tragen nur noch die Ergebnisse ein und Ihre Besucher erhalten sofort die aktuellen Tabellen, Statistiken usw. - Keine Fremdsoftware, keine zeitraubenden Exporte und Uploads mehr!

Es ist ratsam, wenn Sie sich ein Bookmark/Favoriten in Ihrem Browser für den Adminbereich (.../lmoadmin.php) einrichten, damit Sie immer schnell darauf zugreifen können. Im User-Bereich ist aus Sicherheitsgründen kein Link auf den Adminbereich vorhanden (damit niemand auf dumme Gedanken kommt).

Die Besucher können das Skript über die Datei lmo.php starten. Wie man den LMO an seine eigene Homepage anpasst, steht im Abschnitt Anpassung des LMO. Zu jedem Link erhalten Ihre Besucher einen Tooltip, so daß Sie keine weiteren Erklärungen zur Benutzung des Skripts geben müssen.

Der Adminbereich ist, über lmoadmin.php erreichbar. Beim ersten Start melden Sie sich mit dem Namen admin und dem Passwort lmo (Mit L wie Ludwig!) an. Unter Optionen können und sollten Sie das dann sofort ändern, damit kein Unbefugter mehr Zugriff hat.

3. Installation

  1. Update des LMO: Beim Update des LMO 4.0 von älteren LMO4-Versionen (beta/RC1) werden keine Konfigurationsdateien überschrieben, Sie können die Dateien über Ihre vorhandenen Datein kopieren. Führen Sie danach unbedingt die Installation neu aus, indem Sie den LMO aufrufen. Es empfiehlt sich bei Live-Systemen den LMO für den Zeitpunkt der Installation kurz zu sperren. (z.B. über htaccess). Eventuell installierte zusätzliche Addons werden nicht erkannt, diese sind also nach dem Update erneut zu installieren
  2. Das ZIP-Archiv wurde bereits von Ihnen entpackt. Achten Sie darauf, dass die Verzeichnisstruktur beim Entpacken beibehalten wird.
  3. Rufen Sie nun den LMO auf, indem Sie die Adresse zum LMO in Ihren Browser eingeben. Die Installation startet automatisch und sorgt für alle Vorraussetzungen, die der Betrieb des LMO erfordert.
  4. Im Normalfall ist der LMO nun korrekt installiert. Sollte jedoch ein Fehler aufgetreten sein oder Sie können aus anderen Gründen die Installation nicht abschließen, müssen Sie die folgenden Schritte manuell nachvollziehen:

    Manuelle Installation

    • Achtung: Diese Schritte sind nur notwendig, wenn die normale Installation nicht funktioniert!
    • Löschen Sie die Datei install/install.php und kopieren Sie danach den Inhalt des Ordners install über Ihr LMO-Verzeichnis
    • Ändern Sie die Datei config/init-parameters.php mit einem normalen Texteditor und tragen Sie dort die benötigten Pfade ein. Falls die Datei noch nicht existiert, legen Sie diese mit einem normalen Texteditor an.
    • Die Datei sollte so aussehen: <?php
        $lmo_dateipfad='/absoluter/pfad/zum/lmo'; //Dateipfad zum LMO
        $lmo_url='http://www.adresse.ihres/lmo'; //abolute URL zum LMO
      ?>       Ersetzen Sie die beiden markierten Werte mit dem Dateipfad zum LMO-Verzeichnis und mit der Webadresse zum LMO. Falls Sie diese Angaben nicht wissen, fragen Sie bitte Ihren Provider.
    • Nun ändern Sie bei folgenden Datei die Zugriffsrechte (chmod):
      Verzeichnisse:
      777 - das Verzeichnis ligen und alle Unterverzeichnisse
      777 - das Verzeichnis output
      777 - das Verzeichnis config und dessen Unterverzeichnis config/viewer
      777 - das Verzeichnis addon/spieler/stats
      777 - das Verzeichnis addon/tipp/tipps und alle Unterverzeichnisse
      Dateien:
      666 - config/lmo-auth.php
      666 - addon/tipp/lmo-tippauth.txt
      666 - Alle Dateien cfg.txt im Verzeichnis config und dessen Unterverzeichnissen
      666 - alle Ligen im ligen-Verzeichnis und in dessen Unterverzeichnissen
      666 - alle Dateien in den Verzeichnissen output , addon/spieler/stats und tipp/tipps (sofern vorhanden)

    • Damit ist der Liga Manager Online installiert!

4. Login/Logout

Wenn Sie den Adminbereich betreten, sehen Sie zuerst eine Eingabemaske, mit der Sie sich beim System anmelden müssen. Nach der Installation ist der voreingestellte der Nutzername admin und das Passwort lmo (kleines L, kleines M, kleines O) eingestellt. Das sollten Sie aber schnellstmöglich ändern, um Unbefugten den Zutritt zu Ihrem Adminbereich zu verwehren.

Wenn Sie eingeloggt sind, haben Sie rechts oben im Menü immer auch einen Befehl Logout, mit dem Sie sich aus dem Adminbereich ausloggen können. Das sollten Sie der Sicherheit wegen auch immer am Ende einer Sitzung tun.
