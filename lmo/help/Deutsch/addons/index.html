Addons

Für den Ligamanager Online 4 gibt es eine ganze Anzahl an Addons, teilweise sind sie mitgeliefert, teilweise als seperater Download erhältlich. Die Addons erweitern das Grundprogramm um eine Vielzahl von zusätzlichen Funktionen.

Mitgelieferte Addons

Tippspiel-Addon
Erweitert den LMO4 um ein umfangreiches und sehr flexibles Tippspielsystem.
Spielerstatistik-Addon
Einsätze, Tore, Karten - stellen Sie mit diesem Addon Spielerstatistiken zur Verfügung
Minitabellen-Addon
Sie wollen eine verkleinerte Ansicht der aktuellen Tabelle auf ihrer Startseite? Mit diesem Addon ist das problemlos und sehr einfach möglich
LMO-Ticker-Addon
Sie wollen einen Ergebnisticker auf ihrer Startseite? Benutzen Sie dafür das Newstickeraddon
LMO-Classlib (ab LMO4.0 Final mitgeliefert)
Diese Bibliothek bietet zahlreiche Klassen und Methoden für objektorientierte Entwicklungsansätze im LMO, wodurch der Zugriff auf die Informationen einer Ligadatei wesentlich erleichtert wird. Als zentrales Element bindet die classlib darüberhinaus weitere Bibliotheken (z.B. PDF Bibliothek) ein, die dem Entwickler ebenfalls zur Verfügung stehen. Das Modularisierungskonzept der classlib ermöglicht das Auslagern von HTML Code in Funktionen, der somit global zur Verfügung steht und mehrfach verwendet werden kann. Auch wenn Sie äußerlich keine Änderung bemerken, greifen die meisten hier gelisteten Addons auf die Funktionen der Classlib zu.
Mininext-Addon (ab LMO4.0 Final mitgeliefert)
Script, um die nächste Partie einer Liga in einem kleinen Block anzuzeigen. Zusätzlich werden die entsprechenden alten Begegnungen der Mannschaften absteigend sortiert angezeigt.
LMO-Viewer (ab LMO4.0 Final mitgeliefert)
Stellt Ergebnisse und Begegnungen beliebiger Ligen dar und ist dabei äußerst flexibel konfigurierbar. Sie können den Viewer auf beliebigen Seiten ihrer Homepage einbinden.

Weitere Addons

Diese Addons erweitern den LMO um zusätzliche Funktionen. Sie sind nicht mitgeliefert, sondern können separat heruntergeladen und installiert werden.

Dies ist eine unvollständige Liste. Alle erhältlichen Addons sind im Archiv des Supportforum auf www.vest-sport.de zu finden.
